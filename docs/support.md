# Support & Integration

FusionCore is Apache 2.0 and free to use forever. If you need guaranteed results on your specific hardware, two options are available.

---

## Open Source (Free)

- Full source code, Apache 2.0
- `sudo apt install ros-jazzy-fusioncore-ros`
- Community support via [GitHub Issues](https://github.com/manankharwar/fusioncore/issues) (best effort, typically answered within 24 hours)
- Full documentation
